## 🐛 Troubleshooting

### Bot Doesn't Respond
**Check:**
1. Bot token is set in Cloudflare secrets
2. Webhook is set correctly
3. Cloudflare deployment succeeded

### Still Seeing Duplicate Messages
**Solution:**
- Wait a few minutes for deployment to propagate
- Clear Telegram cache
- Try `/start` again

### Copy Button Doesn't Work
**Note:**
- Buttons send instructions, not content directly
- You need to scroll up to copy the actual content
- This is how Telegram works (can't auto-copy)
