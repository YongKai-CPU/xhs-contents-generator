STEP 1: Create GitHub Repository
--------------------------------------------

1. Go to: https://github.com/new
2. Repository name: xhs-contents-generator
3. Make it Public or Private (your choice)
4. DON'T initialize it (no README, no .gitignore)
5. Click "Create repository"


STEP 2: Connect and Push
--------------------------------------------

Copy and paste these commands (replace YOUR_USERNAME with your GitHub username):

# Connect to GitHub
git remote add origin https://github.com/YOUR_USERNAME/xhs-contents-generator.git

# Push your code
git branch -M main
git push -u origin main


STEP 3: Deploy to Railway from GitHub
--------------------------------------------

1. Go to: https://railway.com/project/5daff756-041b-46aa-aca2-a2d79f93b896
2. Click "New" > "Deploy from GitHub repo"
3. Connect your GitHub account
4. Select: xhs-contents-generator
5. Railway will auto-detect and deploy!
