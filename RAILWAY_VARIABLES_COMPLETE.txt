================================================================================
  HOW TO ADD IN RAILWAY
================================================================================

Method 1: One by One (Easiest)
--------------------------------------------
1. Go to Railway Dashboard
2. Click your project
3. Click "Variables" tab
4. Click "New Variable"
5. Paste Name and Value
6. Click "Save"
7. Repeat for all 13 variables


Method 2: Raw Editor (Faster)
--------------------------------------------
1. Go to Variables tab
2. Click "Raw Editor" (top right)
3. Copy ALL variables from "QUICK ADD" section above
4. Paste into Raw Editor
5. Click "Save"


================================================================================
  AFTER ADDING VARIABLES
================================================================================

1. Make sure Build Provider = Dockerfile
2. Add all 13 environment variables
3. Upload Firebase service account JSON file
4. Click "Deploy" or "Redeploy"
5. Wait 3-5 minutes
6. Should deploy successfully!


================================================================================
  TROUBLESHOOTING
================================================================================

Error: Still shows missing variables
Solution: Click "Refresh" in Variables tab, or reload the page

Error: Firebase auth not working
Solution: Make sure to upload the JSON file with: railway upload

Error: Build fails
Solution: Check Build Logs for specific error

Error: App crashes on start
Solution: Check Deployment Logs, verify all variables are set
