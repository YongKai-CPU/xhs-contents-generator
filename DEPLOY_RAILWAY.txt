================================================================================
  Deploy Railway Backend - Complete Guide
================================================================================

STEP 1: Create Railway Service
--------------------------------------------

Option A: Via Web (Easiest)
1. Go to: https://railway.com/project/5daff756-041b-46aa-aca2-a2d79f93b896
2. Click "New" > "Deploy from GitHub repo"
3. Connect your GitHub account
4. Select your repository: xhs contents generator
5. Railway will auto-detect it's a Node.js app
6. Click "Deploy"

OR

Option B: Via CLI
1. In PowerShell, run:
   railway service create
   # Name: xhs-generator-backend
   # Select your project


STEP 2: Link Service
--------------------------------------------

railway service
# Select: xhs-generator-backend

--------------------------------------------

# Example: https://xhs-generator-backend-production.up.railway.app


STEP 7: Update Cloudflare Worker
--------------------------------------------

1. Open: worker/wrangler.toml
2. Line 14: Replace with your Railway URL:
   RAILWAY_BACKEND_URL = "https://your-railway-url.railway.app"
3. Save the file
4. Run: wrangler deploy


STEP 8: Test Everything
--------------------------------------------

# Test Cloudflare Worker
curl https://xhs-generator-worker.yongkailim1.workers.dev/health

# Test end-to-end
curl https://xhs-generator-worker.yongkailim1.workers.dev/health/ready
